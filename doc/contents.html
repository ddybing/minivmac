CONTENTS
of the Mini vMac source distribution

README

Just says to look at the 'doc' folder.
COPYING
The GPL license
doc
Documentation folder. The documentation is in html format, and is an exact copy of what is on "http://www.gryphel.com/c/minivmac/doc" at the time of release.
c_src
Contains all of the C source code files.
lists
Lists of source files. Used by the automatic build scripts.
platform
Contains a folder for each supported platform, including 'mac' for Macintosh and 'win' for Microsoft Windows. See the build documentation for details.
tool
Contains a folder for each supported development tool, including 'mpw' for Macintosh Programmers Workshop, 'mw' for Metrowerks CodeWarrior, 'msvc' for Microsoft Visual C++, and 'bash' for the GNU Project's Bourne Again SHell. See the build documentation for details.

extras

Extra stuff. So far it includes 'mydriver' (source code for a fake sony driver that is patched into the emulated rom), and 'autoquit' (source code for the small program autoquit).
