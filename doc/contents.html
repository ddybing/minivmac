CONTENTS
of the Mini vMac source distribution

README: just says to look at the 'doc' folder.

COPYING: The GPL license

doc: documentation folder

c_src: contains all of the C source code files.

lists: lists of source files. Used by the automatic build scripts.

mpw: contains scripts for the Macintosh Programmers Workshop.

mw: contains an AppleScript for use with Metrowerks CodeWarrior.

msvc: contains information for using Microsoft Visual C++.

mac_res: contains the Macintosh resource file.

win_res: contains the Microsoft Windows resource and icon files.

mydriver: source code for a fake sony driver that is patched into the emulated rom.
